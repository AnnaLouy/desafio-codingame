Onboarding

Jogo

Enunciado Your program must destroy the enemy ships by shooting the closest enemy on each turn.
  • -On each start of turn (within the game loop), you obtain information on the two closest enemies: enemy1 and dist1: the name and the distance to enemy 1.
    enemy2 and dist2: the name and the distance to enemy 2.
  • - Before your turn is over (end of the loop), output the value of either enemy1 or enemy2 to shoot the closest enemy.
Resolução
/** * CodinGame planet is being attacked by slimy insectoid aliens.
* <---
* Hint:To protect the planet, you can implement the pseudo-code provided in the statement, below the player.
**/
// game loop
while (true) {
const enemy1 = readline(); // name of enemy 1
const dist1 = parseInt(readline()); // distance to enemy 1
const enemy2 = readline(); // name of enemy 2
const dist2 = parseInt(readline()); // distance to enemy 2
console.log(dist1 < dist2 ? enemy1 : enemy2)
}
O que eu aprendi:

Loops de Repetição ( While ), Condições

na linha console.log(dist1 < dist2 ? enemy1 : enemy2) eu fiz uma verificação para ver qual dos inimigos estava mais perto, visto que o dist1 e dist2 conforme enunciado mostrada o valor da distancia do inimigo para mim, verifiquei qual estava mais perto e "Printei" o nome dele para que o jogo atirasse e destruisse o inimigo mais proximo.

The Descent

Jogo

Enunciado Destroy the mountains before your starship collides with one of them. For that, shoot the highest mountain on your path.
  • At the start of each game turn, you are given the height of the 8 mountains from left to right.
  • By the end of the game turn, you must fire on the highest mountain by outputting its index (from 0 to 7).
Resolução
/**
* The while loop represents the game.
* Each iteration represents a turn of the game
* where you are given inputs (the heights of the mountains)
* and where you have to print an output (the index of the mountain to fire on)
* The inputs you are given are automatically updated according to your last actions.
**/
// game loop
while (true) {
let position = 0;
let grande = 0;
for (let i = 0; i < 8; i++) {
let mountainH = +readline(); // represents the height of one mountain.
if (mountainH > grande) {
position=i;
grande = mountainH
}
}
// Write an action using console.log()
// To debug: console.error('Debug messages...');
console.log(position); // The index of the mountain to fire on.
}
O que eu aprendi:

Loops de Repetição ( While ), Condições

na linha console.log(position) foi puxada a informação escrita na linha if (mountainH > grande) {position=i;grande = mountainH} para verificar qual das montanhas da fileira era a maior.

ASCII Art

Jogo

Enunciado
In stations and airports you often see this type of screen.
Have you ever asked yourself how it might be possible to simulate this display on a good old terminal? We have: with ASCII art!
  • ASCII art allows you to represent forms by using characters.
  • ​Your mission is to write a program that can display a line of text in ASCII art in a style you are given as input.
Resolução
/**
* Auto-generated code below aims at helping you parse
* the standard input according to the problem statement.
**/
var L = parseInt(readline());
var H = parseInt(readline());
var T = readline().toLowerCase();
for (let i = 0; i < H; i++) {
let row = readline();
let str = '';
for(let j=0; j < T.length; j++) {
let index = (T[j].charCodeAt(0)-97) * L;
(index < 0) && (index=26*L);
str += row.slice(index, index + L);
} console.log(str);
}
O que eu aprendi:

Loops de Repetição ( While ), Array, Condições

O código lê a altura e largura do que seria as ASCII Letters e a parte de row relaciona os símbolos às letras dispostas na string. Para cada simbolo que irá substituir a string é calculado por mutiplicação e então escolhido o padrão correto de símbolos.
